# Enterprise RAG Knowledge Assistant

A production-grade Retrieval-Augmented Generation (RAG) Knowledge Assistant built with FastAPI, PostgreSQL, Qdrant, Redis, Celery, React, and Docker.

## Planned Features

- JWT Authentication
- Document Upload
- PDF/DOCX/TXT Processing
- Semantic Search
- Vector Database
- Retrieval-Augmented Generation
- Conversation History
- Admin Dashboard
- Background Document Indexing
- Docker Deployment

## Tech Stack

- FastAPI
- PostgreSQL
- Qdrant
- Redis
- Celery
- React
- Tailwind CSS
- Docker
